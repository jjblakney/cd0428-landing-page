# Landing Page Project
Manipulating the DOM exercise.Exercise programmatically builds navigation,
scrolls to anchors from navigation,
and highlights section in viewport upon scrolling.

## Instructions
Scroll through the sections. When they are active in viewport the section will be highlited as well as the matching section in the navbar. Click a navabr item. Smooth scrolling should occur and you will be taken to the matching section 

## Dependencies: None
